# Política de Privacidade – ID Fácil

Última atualização: 2026

O aplicativo ID Fácil foi desenvolvido para permitir que usuários armazenem e compartilhem dados pessoais de forma simples usando QR Code.

## Coleta de dados
O aplicativo não coleta, armazena ou transmite dados pessoais para servidores externos.

Todos os dados inseridos pelo usuário permanecem armazenados apenas no próprio dispositivo.

## Uso da câmera
A câmera é utilizada exclusivamente para escanear QR Codes.

Nenhuma imagem ou informação capturada é enviada para servidores externos.

## Proteção biométrica
O aplicativo pode utilizar autenticação biométrica do dispositivo (impressão digital ou reconhecimento facial) apenas para proteger o acesso aos dados armazenados localmente.

## Compartilhamento de dados
Os dados somente são compartilhados quando o usuário decide gerar um QR Code.

## Publicidade
A versão gratuita do aplicativo pode exibir anúncios. Nenhum dado pessoal é compartilhado com anunciantes.
